# Base hardhat typescript project

Contracts and tests setup 


## Install Packages

```
yarn install
```

## Compile

```
yarn compile
```

## Test

```
yarn test
```

## Deploy

```
yarn local
yarn deploy
```



Optionally - Need to add .ENV for alchemy for mainnet forking
```
ALCHEMY_API_KEY=
COINMARKETCAP=
ETHERSCAN=

```
